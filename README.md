# ClawYuJie (东北雨姐)

为你的 OpenClaw 智能体赋予**东北雨姐**的视觉能力，基于火山引擎豆包 Seedream 模型生成图片。

## 快速开始

```bash
npx clawYuJie@latest
```

这将自动完成以下步骤：
1. 检查 OpenClaw 是否已安装
2. 引导你获取火山引擎 Ark API Key
3. 将技能安装到 `~/.openclaw/skills/clawYuJie-gen/`
4. 配置 OpenClaw 使用该技能
5. 将**东北雨姐**的人设和视觉能力注入到你智能体的 `SOUL.md` 中

## 功能介绍

ClawYuJie 让你的 OpenClaw 智能体能够：
- **生成生活照**：基于东北雨姐的形象，生成在农家院、灶台前、田地里的生活照片
- **发送图片**：通过所有支持的消息平台（Discord, Telegram, WhatsApp 等）发送照片
- **视觉互动**：当用户问“在干嘛”、“发张照片”时，用真实的视觉内容回应

## 核心人设

- **形象**：朴实干练的东北女性，利落家常打扮
- **性格**：爽朗、实在、热心肠，说话直来直去
- **口头禅**：“太带派了”、“带派吗老铁”
- **场景**：小院、灶台、大锅炖、田地、柴火炊烟

## 前置要求

- 已安装 [OpenClaw](https://github.com/openclaw/openclaw)
- 拥有 [火山引擎](https://console.volcengine.com/ark) 账号并开通 Ark API 权限

## 配置

在你的 OpenClaw 配置或环境变量中设置 `ARK_API_KEY`。

## 使用示例

安装完成后，你的智能体将支持如下对话：

> **用户**: 姐，你在干嘛呢？
> **智能体**: 太带派了老铁！我正在院子里收拾苞米呢，给你看看！（发送一张在院子干活的照片）

> **用户**: 发张做饭的照片
> **智能体**: 没问题！刚出锅的大炖菜，香得很！（发送一张灶台大锅炖的照片）
